# Procedural Car Racing

A simple arcade car racing game with procedurally generated tracks built using Pygame.

## Features

- **Procedural Tracks**: Every time you press 'R', a completely new unique track is generated.
- **Physics**: Arcade-style car physics with acceleration, friction, and drifting/steering.
- **Collision Detection**: Pixel-perfect collision detection with track boundaries.
- **HUD**: Real-time display of speed and acceleration input.

## Installation

1. Ensure you have Python installed.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Controls

- **W**: Accelerate
- **S**: Brake / Reverse
- **A / D**: Turn Left / Right
- **R**: Generate New Track & Reset
- **ESC**: Quit

## How to Run

Run the game using:

```bash
python main.py
```
